# Cafe Sales: Data Cleaning & Exploratory Analysis

## Project Objective
The goal of this project was to take a "dirty" retail dataset and build a reproducible pipeline to clean, validate, and analyze sales performance.

## Key Technical Solutions
- **Mathematical Validation:** Fixed inconsistencies where `Total Spent` did not match `Quantity * Price`.
- **Data Imputation:** Replaced 'UNKNOWN' and 'ERROR' tags with statistically sound placeholders.
- **Trend Analysis:** Visualized seasonal revenue spikes to assist in staff and resource planning.
- **Cross-Sectional Insights:** Used Heatmaps to analyze item popularity across different store locations.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib & Seaborn
- Jupyter Notebook
